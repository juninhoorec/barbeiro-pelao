# 💈 Barbeiro Pelão - Projeto Front-End

Este é um projeto de site de barbearia desenvolvido com **HTML, CSS e JavaScript**, focado em criar uma experiência moderna, elegante e responsiva.

---

## ✨ Demonstração

📌 O projeto simula um site profissional de barbearia com:

- Página inicial moderna (Hero)
- Seção de serviços
- Sobre a barbearia
- Contato
- Botões interativos
- Design escuro e sofisticado

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (básico/interações)
- Google Fonts
- Design responsivo

---

## 🎨 Estilo do projeto

O site segue um estilo:

- Minimalista
- Premium (tons escuros + dourado)
- Inspirado em barbearias modernas
- Foco em experiência visual
